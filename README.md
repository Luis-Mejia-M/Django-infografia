# Django-infografia
# Proyecto Django – Gestión de Artículos

Este proyecto fue desarrollado como parte de la actividad "Infografía Django".
Este implementa un sistema de gestión de artículos utilizando el framework Django.

## Objetivo

Demostrar el proceso completo de:

- Creación de un entorno virtual
- Creación de un proyecto Django
- Configuración de base de datos (SQLite)
- Creación y registro de una aplicación
- Implementación de modelos
- Migraciones
- Configuración de URLs
- Creación de vistas (ListView y CreateView)
- Desarrollo de templates
- Generación y visualización de entidades

## Funcionalidades Implementadas

- Visualización de todos los artículos
- Creación de nuevos artículos
- Uso de migraciones para sincronizar modelo y base de datos
- Manejo de URLs dinámicas
- Renderizado mediante templates

## Tecnologías utilizadas

- Python 3
- Django 6
- SQLite
- HTML
